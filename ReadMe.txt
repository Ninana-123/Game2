1)Movement 
-Arrow keys to move left right up and down

2)Rotation
-To Rotate the obejct press "T" for clockwise rotation and "R" for anti-clockwise rotation

3)Scaling
-To scale the object press "Z" scale up and "X" is to scale down 


4)Imgui

- To select different entities
- entity 0 is the background, 
 -entity 1-3 is the different game objects 
Click create entity to create a base entity with no tex or component loaded
Click clone to clone currently selected entity

5) Pathfinding ECS
- A sample implementation is to add the current lines into the testscene.txt as follows, where "Pathfinding is the flag for it to trigger the pathfinding ecs, startX and Y is the starting point and goal X and Y are the endpoints:
Pathfinding
startX: 0
startY: 0
goalX: 200
goalY: 200
EndEntity
